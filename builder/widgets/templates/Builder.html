About dgrid builder

This application allows you to experiment with many of dgrid's features.

  1. Choose and configure mixins and extensions in the Features grid on the left
  2. Create, reorder, and configure columns in the Columns grid in the middle
  3. As you make changes, the pane on the right will update to display a grid based on your configuration. The Source Code tab displays the code required to achieve the chosen configuration.

Note that while this application demonstrates many of dgrid's features, it does not demonstrate all of them. Namely, it does not include a way to configure advanced column structures, such as sub-rows, column sets, and compound columns.
